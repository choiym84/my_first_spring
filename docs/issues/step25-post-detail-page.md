# [Step 25] Create Post Detail Page

## Goal

게시글 상세 조회 화면을 만든다.

## Tasks

- [ ] `GET /posts/{id}` 구현
- [ ] `postDetail.html` 생성
- [ ] id로 게시글 조회
- [ ] 없는 게시글 요청 처리
- [ ] 목록 페이지에서 상세 페이지로 이동 링크 추가

## Acceptance Criteria

- [ ] 게시글 제목을 클릭하면 상세 화면으로 이동한다.
- [ ] 상세 화면에서 title, content, writer, createdAt 이 보인다.
- [ ] 존재하지 않는 id 요청이 적절히 처리된다.

## Notes

- 처음에는 예외 메시지 화면 또는 목록 redirect로 단순 처리해도 된다.
